Github 预览效果最好

代码效果在

https://github.com/johony/oss/blob/master/test/2025-06-21-table-code-preview.md

打开后以看到多行代码在表格中正常展示出来。

对比会发现，azure 必须加入代码项目才能保存暗黑主题的设置。如果微软帐号没有加入项目，刷新页面之后页面又回到亮色模式。

Github 公开仓库久没有这个问题，即使不登陆帐号，也可以在暗色模式下看markdown文件。

另外 azure 并不支持 GitHub Style Alert 样式。
