OpenCV起手式

编译环境(Mac OSX)

Mac OSX安装了Xcode, 然后去CMake 下载Mac OSX的安装包, 版本选择Latest Release(稳定版)。

下载OpenCV源代码

进入OpenCV官网, 最后一行看到The source code for all platforms can be downloaded from GitHub: zip and tar.gz这行, 点击下载源代码。

下载后的源代码解压缩, 进入代码根目录, 新建一个 build 文件夹。

配置Makefile&编译

打开CMake , Browse Source选择源代码的根目录, Browse Build选择刚才新建的 build 文件夹。

点击下面的 Configure , 选择Unix Makefiles, 选中’Use default native compiles’, 点击Done , 接着点击 Generate 这一步生成Makefiles文件。

编译

进入 build 文件夹, 终端输入 make, 之后输入 sudo make install 。好了, 现在就开始编译了 , 编译完成后会看到终端结尾如下提示。

[100%] Linking CXX executable ../../bin/opencv_version
[100%] Built target opencv_version

进入到 bin 目录, 就可以看到编译后的可执行程序。
